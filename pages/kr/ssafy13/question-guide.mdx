# SSAFY 13기 디스코드 질문 매뉴얼

안녕하세요 레드브릭 디스코드 대응팀입니다.

## 📌 질문할 때 주의해주세요!

### 🛠️ 코드 작동이 안될 때

1. 코드를 다시 한번 확인해주세요
2. 질문 시 아래 내용을 모두 포함하여 질문해 주세요
   - 어떤 것을 구현하고 싶은지
   - 어떻게 안되는지 (에러 발생했다면 에러 내용도 같이)
   - 에셋 이름, 안되는 코드를 복사하여 공유
   - 화면 캡쳐 첨부 (코드는 캡쳐보다 복사 후 붙여넣기로 전달 해주시는게 해결에 용이합니다)
