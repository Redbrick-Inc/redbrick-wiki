# Data

> 데이터 탭은 객체에 대한 정보를 표시합니다.

## Title

객체는 각각 고유한 제목을 가져야 합니다. 동일한 제목의 여러 객체가 있으면 제대로 작동하지 않거나 설정대로 작동하지 않을 수 있으므로 이를 권장합니다.

## Class

객체가 속한 클래스에 대한 정보를 나타냅니다. 기본적으로 이 값은 변경할 수 없습니다.

## Name

객체가 가진 이름을 표시합니다. 기본적으로 이 값은 변경할 수 없습니다.

## Parent

객체보다 한 단계 위에 있는 요소를 나타내는 값입니다.
