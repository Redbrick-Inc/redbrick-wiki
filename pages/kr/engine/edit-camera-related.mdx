# 편집 카메라 조작 기능

> 편집 카메라를 선택한 오브젝트로 이동시키거나  
선택한 오브젝트의 위치/회전값을 편집 카메라의 위치/회전값과 일치시킵니다.

---

### 선택한 오브젝트로 편집 카메라 이동시키기

Explorer 탭 또는 Scene 뷰에서 오브젝트를 선택하고 `Shift + F`를 누르면 편집 카메라가 선택된 오브젝트로 이동할 수 있습니다.

### 선택한 오브젝트의 위치/회전 값을 편집 카메라와 일치시키기

Explorer 탭 또는 Scene 뷰에서 오브젝트를 선택하고 `Cmd + Shift + F`를 누르면 선택한 오브젝트의 위치/회전값을 편집 카메라와 일치시킬 수 있습니다.

    </p>
</center>
<Callout type="info">
    해당 기능을 카메라 오브젝트에 적용하면 손쉽게 카메라 구도를 잡을 수 있습니다  
    여러 카메라를 배치시키고 카메라 오브젝트의 `.activate()`를 통해 카메라 연출을 할 수 있습니다.
