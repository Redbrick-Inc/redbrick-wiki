# 에셋 교체하기

> 에셋 교체에 대한 정보를 제공합니다.

---

### 에셋 종류

에셋은 크게 두 종류가 있습니다.

1. **3D 오브젝트**: Camera, Spawn, Light, Effect, Mesh, Sound, Video와 같은 오브젝트가 여기에 속합니다. 
<br/>Transform 속성으로 position, rotation, scale을 가지고 있습니다.

2. **GUI 오브젝트**: GUI 오브젝트가 여기에 속합니다. 
<br/>Transform 속성으로 center, anchor, offset, size를 가지고 있습니다.

<br/>
<center>![replace-asset_asset-type](/images/engine/replace-asset/replace-asset_asset-type.png) 에셋 종류</center>

<Callout type="error">
    서로 다른 종류의 OBJECT끼리 교체할 수 없습니다.
    <br/>현재 Sound와 Video는 다른 모든 종류의 에셋과 교체할 수 없습니다. 추후 추가될 예정입니다.
</Callout>
<br/>

---

### 에셋 교체

먼저 에셋 교체 시 유지되는 속성은 다음과 같습니다.
- 공통
    - title(제목)
    - Parent(부모 오브젝트)
- 3D 오브젝트
    - Object Transform
- GUI 오브젝트
    - GUI Transform: anchor, offset(px)
    - Text: textInput, fontSize, fontColor

<br/>
<center>![replace-asset_retained-property](/images/engine/replace-asset/replace-asset_retained-property.png) 에셋 교체 시 유지되는 속성</center>

<br/>
### 교체 순서
<Steps>
    ### 교체 할 에셋 선택하기

    Explorer 탭 또는 Scene 패널에서 교체하고 싶은 에셋을 선택할 수 있습니다.

    <br/>
    <center>![replace-asset_step-1](/images/engine/replace-asset/replace-asset_step-1.png) 교체하고 싶은 에셋 선택</center>

    ### 교체될 에셋 더보기(...) 클릭하기

    에셋 라이브러리 또는 My 에셋에 교체될 에셋을 선택하고, 해당 에셋의 더보기를 눌러줍니다.

    <br/>
    <center>![replace-asset_step-2](/images/engine/replace-asset/replace-asset_step-2.png) 에셋의 더보기 클릭</center>

    ### Replace를 눌러 에셋 교체하기
    교체가 완료되었습니다! 만약 교체가 되지 않는다면, <Link href="#에셋-종류"><u>해당 내용</u></Link>을 참고해줍니다.

    <br/>
    <center>![replace-asset_step-3](/images/engine/replace-asset/replace-asset_step-3.png) Replace 클릭</center>
</Steps>
