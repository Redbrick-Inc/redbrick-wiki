# 유니티 제출 방법
<br />
<Callout>

아래 유니티 프로젝트 제출 방법은 <span style={{ color: 'white' }}>**영상**</span>으로도 소개되어 있습니다. <br />

<br/>
<YouTube youTubeId="AsOIFQIg_yU" />

위의 영상을 보시면 아래 내용들을 더욱 자세하게 참고하실 수 있습니다!
만약에 레드 브릭 사이트에 프로젝트 publish 까지 완료 하셨다면, 해당 글에서 <span style={{ color: 'white' }}>**3단계 : 프로젝트 제출하기**</span> 로 바로 보셔도 됩니다.

</Callout>

## 🛠️ 단계 1 : Unity에서 WebGL 파일 빌드하기

<Steps>

### Unity 프로젝트 열기

Unity를 열고 WebGL로 내보낼 프로젝트를 불러옵니다.
<br />
<center>![IMAGE](/images/connect2024/image.png)</center>

---

### Build Settings로 이동

상단 메뉴 바에서 **File**을 클릭하고 **Build Settings**를 선택합니다.
<br />
<center>![IMAGE](/images/connect2024/image_(1).png)</center>

---

### WebGL Platform 선택

**Build Settings** 창에서:
- **Platform** 섹션에서 **WebGL**을 선택합니다.
- WebGL이 보이지 않으면 **Open Download Page**를 클릭하여 설치합니다.
- **Switch Platform**을 클릭하여 WebGL로 전환합니다.
<br />
<center>![IMAGE](/images/connect2024/image_(2).png)</center>

---

### Player Settings 조정 (선택 사항)

빌드를 맞춤 설정 하려면:
- **Build Settings** 창에서 **Player Settings**를 클릭합니다.
<br />
<center>![IMAGE](/images/connect2024/image_(3).png)</center>

- **Resolution and Presentation, Publishing Settings** 등의 설정을 필요에 맞게 조정할 수 있습니다.
<br />
<center>![IMAGE](/images/connect2024/image_(4).png)</center>

---

### 🔑  주의 사항

<Callout type="warning">
  1. Player Settings에서 <span style={{ color: 'white' }}>**Compression Format**</span>이 disable 되어 있으면 WebGL 파일이 안나오므로 , <span style={{ color: 'white' }}>**Brotli**</span> 로 변경해야 합니다. 
  2. Player Settings에서 <span style={{ color: 'white' }}>**Decompression Fallback이 활성화**</span>되어 있는지 확인하여 WebGL 빌드가 온라인에서 원활하게 작동하도록 하세요.

  <center>![IMAGE](/images/connect2024/image_(5).png)</center>

  또한, <span style={{ color: 'white' }}>**development build**</span> 체크를 해제 했는지 확인합니다. ( 체크되어 있으면 오류납니다. )  
  <br />
  <center>![IMAGE](/images/connect2024/image_(6).png)</center>
</Callout>

---

### 🚫 빌드 버튼 활성화 안되는 오류 해결하기

_빌드 버튼 활성화 안되는 오류 해결하는 방법을 알고 싶다면 아래 글을 읽어주세요._

<Callout type="error">
  1. 아래와 같은 오류가 발생하면서 빌드 버튼이 활성화되지 않을 경우

  <center>![IMAGE](/images/connect2024/image_(7).png)</center>

</Callout>

<Callout type="error">
  2. Project Setting 에 들어가서 제일 오른쪽에 있는 탭을 눌려주세요. <br /> 그 후, Other Settings 를 눌려서 페이지를 펼쳐주세요.

  <center>![IMAGE](/images/connect2024/image_(8).png)</center>

</Callout>
<Callout type="error"> 
  3. Color Space 에서 Linear 라고 되어 있는 부분을 Gamma 라고 바꿔주세요.

  <center>![IMAGE](/images/connect2024/image_(9).png)</center>

  4.다음과 같이 설치 창이 뜨면 **change to Gamma** 버튼을 눌려주세요.
<br />
  <center>![IMAGE](/images/connect2024/image_(10).png)</center>

  </Callout>
  <Callout type="error">
  5. 아래와 같이 정상적으로 바뀌고 오류가 없어지면서 Build 버튼이 활성화된 것을 확인할 수 있습니다.

  <center>![IMAGE](/images/connect2024/image_(11).png)</center>

</Callout>

---

### 프로젝트 Build 하기

Build Settings 창에서 Build 버튼을 클릭합니다. 
Save as 옆에는 폴더 이름을 팀이름_게임이름 으로 작성해주세요. <br /> 그 이후, Save 버튼을 눌려주세요.
<br />
  <center>![IMAGE](/images/connect2024/image_(12).png)</center>

아래 where 옆에 있는 주소로 파일이 저장됩니다.
<br />
  <center>![IMAGE](/images/connect2024/image_(13).png)</center>

저장 버튼을 누르면 Unity가 WebGL 형식으로 프로젝트를 빌드하기 시작합니다.
<br />
  <center>![IMAGE](/images/connect2024/image_(14).png)</center>

---

**📂 WebGL 빌드가 준비되었습니다!** 선택한 폴더에서 파일을 찾을 수 있으며, 이제 온라인 업로드가 가능합니다.

</Steps>

## 🛠️ 단계 2 : Unity WebGL 파일 Redbrick에 업로드하기

<Steps>

### Redbrick 사이트 열기

다음 사이트 (https://creator.redbrick.land/) 를 엽니다.  

---

### 회원가입 및 로그인

프로젝트를 올리기 위해서는 계정이 필요하며 어떤 계정으로 회원 가입 하든 상관없습니다.
회원 가입 후 로그인을 해줍니다.
<br />
  <center>![IMAGE](/images/connect2024/image_(15).png)</center>

---

### Import 버튼 찾기

계정 옆에 있는 Create 버튼을 누릅니다.
<br />
  <center>![IMAGE](/images/connect2024/image_(15)-1.png)</center>

사이트 왼쪽 패널에서 Import 버튼을 찾습니다.
<br />
  <center>![IMAGE](/images/connect2024/image_(16).png)</center>
  <center>![IMAGE](/images/connect2024/image_(17).png)</center>

---

### WebGL 파일 선택하기

**Import** 버튼을 클릭하여 WebGL 파일이 저장된 폴더로 이동합니다. <br />
위에서 저장한 파일 폴더를 누른 후 업로드 버튼을 클릭합니다.
<br />
  <center>![IMAGE](/images/connect2024/image_(18).png)</center>

---

### 프로젝트 게시 준비

위에서 업로드 버튼을 누르면 아래와 같은 창이 뜹니다.  <br />
**이름**, **설명**, **썸네일 이미지**를 입력을 다 한 후, **Publish** 버튼을 클릭 합니다.
<br />
  <center>![IMAGE](/images/connect2024/image_(19).png)</center>

그럼, 프로젝트가 Redbrick에 업로드되었습니다! <br />
publishing 탭을 클릭해서 올린 프로젝트를 클릭 하세요. 그럼 온라인에서 확인하고 공유할 수 있습니다.
<br />
  <center>![IMAGE](/images/connect2024/image_(20).png)</center>
  <center>![IMAGE](/images/connect2024/image_(21).png)</center>

</Steps>
