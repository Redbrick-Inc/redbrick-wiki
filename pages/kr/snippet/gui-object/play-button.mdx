  왼쪽 상단의 + 버튼을 클릭 > GUI > 원하는 시작 버튼 선택

  ### 코드

  <Tabs items={['씬 스크립트', '오브젝트 스크립트']} defualtIndex="0">
    <Tabs.Tab>
      씬 스크립트로 작성하면 아래와 같이 작성할 수 있습니다.

      ```js showLineNumbers copy filename="PresetScript"
      const startbtn = GUI.getObject("ballstopcle_button_play_on(e77)");

      function Start() {
        startbtn.onClick(() => {
          startbtn.hide();
        });
      }
      ```
    </Tabs.Tab>
    <Tabs.Tab>
      오브젝트 스크립트로 작성한다면 다음과 같이 작성할 수 있습니다.

      ```js showLineNumbers copy filename="ballstopcle_button_play_on(e77)"
      function Start() {
        this.onClick(() => {
          this.hide();
        });
      }
      ```
    </Tabs.Tab>
  </Tabs>
