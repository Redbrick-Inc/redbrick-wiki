# 아이템 수집하기

<Steps>

### 원하는 곳에 객체를 배치하세요.

    <Callout type="info">
        동일하거나 비슷한 역할을 하는 오브젝트의 경우, 타이틀에 숫자를 붙여 순서대로 이름을 지으면 코딩 할 때 좀 더 편리해집니다.
    </Callout>

    <Callout type="error" icon="‼️">
    `onCollide` 같은 메소드를 사용하려면 객체의 속성 패널에서 Body를 체크하는 것을
    잊지 마세요!

    </Callout>

### 코드

<Tabs items={["예시 1", "예시 2"]} defaultIndex="0">
    <Tabs.Tab>
        <Callout type="info">
            이것은 기초적인 예제입니다. 더 개선된 해결책을 원하시면 예시 2를 참조하세요 👉🏼
        </Callout>
        ```js showLineNumbers copy
        const diamond0 = WORLD.getObject("diamond0");
        const diamond1 = WORLD.getObject("diamond1");
        const diamond2 = WORLD.getObject("diamond2");
        const diamond3 = WORLD.getObject("diamond3");
        const diamond4 = WORLD.getObject("diamond4");

        function Start() {
                diamond0.onCollide(PLAYER, () => {
                    diamond0.kill();
                });
                diamond1.onCollide(PLAYER, () => {
                    diamond1.kill();
                });
                diamond2.onCollide(PLAYER, () => {
                    diamond2.kill();
                });
                diamond3.onCollide(PLAYER, () => {
                    diamond3.kill();
                });
                diamond4.onCollide(PLAYER, () => {
                    diamond4.kill();
                });
        }
        ```
    </Tabs.Tab>
    <Tabs.Tab>

        이것은 중복된 줄을 최소화하여 조금 더 개선된 코드입니다.

         ```js showLineNumbers copy
        function Start() {
            // 5 diamonds
            for (let i = 0; i < 5; i++) {
                const diamond = WORLD.getObject("diamond" + i);

                diamond.onCollide(PLAYER, () => {
                    diamond.kill();
                });
            }
        }
        ```
    </Tabs.Tab>

</Tabs>
