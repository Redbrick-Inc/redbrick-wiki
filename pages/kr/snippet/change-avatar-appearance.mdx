# 아바타 모습 바꾸기

<center>
<br/>
![](../../../images/change-avatar.gif)
</center>

플레이어 아바타의 모습을 안보이게 하고, 다른 오브젝트를 플레이어의 자식으로 추가하여 모습이 바뀐것처럼 연출할 수 있습니다.  

<Callout type="warning">
  모습을 바꿀 아바타를 미리 씬에 추가해 놓아야 합니다.
</Callout>

```js showLineNumbers copy filename="change_avatar"
const obj = WORLD.getObject("obj");  //모습을 바꿀 오브젝트
const btn = GUI.getObject('btn');

btn.onClick(function() {
    PLAYER.children[0].visible = false; //플레이어 아바타 감추기
    PLAYER.add(obj); //오브젝트를 플레이어의 자식으로 넣음
    obj.position.set(0, 0.5, 0); //오브젝트 위치 설정
})
```
