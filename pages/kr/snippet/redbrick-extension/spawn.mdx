# 시작점 설정하기

> 완성된 게임에서 아바타 출현 위치를 설정하는 방법에 대한 지침을 제공합니다.

게임이 완성되면, 마지막으로 아바타의 출현 위치를 설정해야 합니다.

<Callout type="info">
  원래 위치로 돌아가는 방법을 찾고 있고 지침이 필요하다면, 이
  [가이드라인](/kr/snippet/redbrick-extension/respawn)을 따라해보세요.
</Callout>

<Steps>
### Starting Point 가져오기

- 에셋 라이브러리 > 스폰 메뉴에서 Starting Point를 가져옵니다.

<br />
<center>![spawn_starting-point](/images/snippet/redbrick-extension/spawn/spawn_starting-point.png) 시작점</center>

### Starting Point 배치하기

- 아바타가 나타날 위치에 Starting Point를 배치하세요.
- 이때, Starting Point의 변환 속성 중 회전 속성을 변경하여 아바타가 나타날 때의 시야각을 설정할 수 있습니다.

<br />
<center>![spawn_starting-point_deploy](/images/snippet/redbrick-extension/spawn/spawn_starting-point-deploy.png) Starting Point 배치하기</center>

### 플레이

- 플레이 버튼을 누르고 게임을 테스트하세요.

<br />

  <p style={{margin: '0'}}>플레이 모드</p>
</center>
</Steps>
