# 시작하기

> 전체 스크립팅 접근 방법에 대한 안내를 제공합니다.

각 객체에 대해 속성 패널에서 스크립트를 생성하고 작성할 수 있습니다.

스크립트는 클라이언트에서 실행되는 클라이언트 스크립트와 서버에서 실행되는 서버 스크립트로 나뉩니다.

스크립트 작성을 돕기 위해 이벤트 함수, 전역 객체 및 다양한 API를 제공합니다.
