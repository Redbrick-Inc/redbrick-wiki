# MathUtils

> 수학 함수들을 제공하는 모듈입니다. `MathUtils.abs() ` 와 같은 형태로 사용할 수 있습니다.

### .abs()

`.abs(number) `

주어진 수의 절댓값을 반환합니다.

### .round()

`.round(number, digit) ` 

주어진수를 반올림하여 값을 반환합니다. digit은 소수점 아래 몇 번째 자리까지 나타내는지를 의미합니다.

### .ceil()

`.ceil(number, digit) `

주어진수를 올림하여 값을 반환합니다. digit은 소수점 아래 몇 번째 자리까지 나타내는지를 의미합니다.

### .floor()

`.floor(number, digit) `

주어진수를 내림하여 값을 반환합니다. digit은 소수점 아래 몇 번째 자리까지 나타내는지를 의미합니다.

### .pow()

`.pow(number, exponent) `

주어진 수를 밑으로 exponent를 지수로 하여 계산한 값을 반환합니다.

### .sqrt()

`.sqrt(number) `

주어진 수의 제곱근 값을 반환합니다.

### .log()

`.log(number, base) `

base를 밑으로 하여 계산한 로그값을 반환합니다.

### .getRandom()

`.getRandom(min, max) `

min, max 범위 내의 무작위 정수를 반환합니다.
