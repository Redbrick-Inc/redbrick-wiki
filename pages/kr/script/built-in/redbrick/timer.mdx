# Timer

> 시간을 측정하는 타이머 기능을 제공하는 모듈.

## 메소드들

### .start()

타이머를 시작하는 함수.

### .pause()

타이머를 중지하는 함수.

### .resume()

정지 상태인 타이머를 다시 시작하는 함수.

### .reset()

타이머를 초기화하는 함수.

### .getTime()

`.getTime()` : float

호출 시의 타이머 시간 값을 반환합니다. 값의 단위는 초(sec)입니다.

> Example

```js copy
const timer = new REDBRICK.Timer();

timer.start();
timer.pause();
timer.resume();
timer.reset();

const currentTime = timer.getTime();
console.log(currentTime);
```
