# Rank

> 플레이어의 점수를 저장하고 랭킹(리더보드)을 보여줄 수 있는 모듈입니다.

---

각 게임별로 플레이어의 점수를 저장하고 랭킹을 보여주는데 사용되는 클래스입니다.

<br/>
<center>

<Callout type="info">
  현재는 랭킹 UI가 한 종류 밖에 없지만, 추후 추가될 예정입니다.
</Callout>

## 메소드

### `.saveScore(score)`

플레이어의 점수를 서버에 저장합니다.

<Callout type="warning">
    score는 **정수**로 변환되어 저장되므로, 점수를 설계할 때 **정수**로 계산되도록 고려해야 합니다.
</Callout>

### `.show()`

랭킹을 보여줍니다. 최상위에 현재 플레이하고 있는 플레이어의 점수가 표시됩니다.

### `.hide()`

랭킹을 숨깁니다. 랭킹이 보여지면 닫기(X)를 눌러서 끄기 전까지 다른 GUI가 클릭되진 않지만, `setTimeout()`을 사용하여 몇 초 뒤에 자동으로 끄고 싶다면 사용할 수 있습니다.

## 예시

```js showLineNumbers copy filename="rank-button" {3}
// This Button shows the ranking
this.onClick(() => {
    REDBRICK.Rank.show();
});
```

```js showLineNumbers copy filename="game-manager" {4}
// This Function saves the score as Integer.
function onGameEnd() {
    // ...
    REDBRICK.Rank.saveScore(score);
}
```

<br/>
`.hide()`를 아래와 같이 사용할 수 있습니다.

```js showLineNumbers copy filename="rank-button_auto-hide" {3,5}
// This Button shows the ranking and hides after 1 sec
this.onClick(() => {
    REDBRICK.Rank.show();
    setTimeout(() => {
        REDBRICK.Rank.hide();
    }, 1000);
});
```
