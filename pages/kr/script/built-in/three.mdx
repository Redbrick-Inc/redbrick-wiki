# THREE

> 웹 브라우저에서 3차원 그래픽을 구현하기 위해 사용하는 자바스크립트 라이브러리입니다.

레드브릭 스튜디오는 THREE를 기반으로 만들어져 있으며 스크립트 내에서도 THREE에 접근하여 동일한 기능들을 사용할 수 있습니다. API등을 통해서 추가적으로 제공하는 기능 외에도 THREE클래스들에 존재하는 속성과 함수들을 사용해 본다면 보다 자유롭게 3D공간을 창조하고 완성도 있는 스크립트를 작성할 수 있게 될 것입니다.
