# DateUtils

> 날짜와 시간을 다루는 함수들을 제공하는 모듈입니다. `DateUtils.getYear()` 와 같은 형태로 사용할 수 있습니다.

### .getYear()

현지 시간 기준 연도를 반환합니다.

### .getMonth()

현지 시간 기준 월을 반환합니다. number값으로 반환되며 0부터 시작됩니다. 

`ex) 1월 = 0, 3월 = 2`

### .getDate()

현지 시간 기준 일을 반환합니다.

### .getDay()

현지 시간 기준 요일을 반환합니다. number값으로 반환되며 일요일부터 0으로 시작합니다.

`ex) 일요일 = 0, 토요일 → 6`

### .getHours()

현재 시점에서의 시를 반환합니다.

### .getMinutes()

현재 시점에서의 분을 반환합니다.

### .getSeconds()

현재 시점에서의 초를 반환합니다.
