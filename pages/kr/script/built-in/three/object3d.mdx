# Object3D

> 3D 공간의 대부분의 객체가 상속받는 클래스입니다.

## Properties

`.children` : 배열

객체의 자식이 있는 배열입니다. 개체를 수동으로 그룹화하는 방법에 대한 자세한 내용은 그룹을 참조하세요.

`.parent` : Object3D

장면 그래프에 있는 개체의 부모입니다. 객체는 최대 하나의 상위를 가질 수 있습니다.

`.castShadow` : 부울

객체가 그림자 맵으로 렌더링되는지 여부입니다. 기본값은 거짓입니다.

`.receiveShadow` : 부울

재질이 그림자를 받는지 여부입니다. 기본값은 거짓입니다.

`.position` : 벡터3

개체의 로컬 위치를 나타내는 Vector3입니다. 기본값은 '(0, 0, 0)'입니다.

`.rotation` : 오일러

라디안 단위의 객체의 로컬 회전(오일러 각도 참조)입니다.

`.scale` : 벡터3

객체의 로컬 스케일입니다. 기본값은 Vector3( 1, 1, 1 )입니다.

`.quaternion` : 쿼터니언

쿼터니언으로서의 객체의 로컬 회전.

`.visible` : 부울

true인 경우 객체가 렌더링됩니다. 기본값은 true입니다.

## Methods

`.add ( object : Object3D, ... )` : 이

개체를 이 개체의 하위로 추가합니다. 임의 개수의 개체를 추가할 수 있습니다. 개체는 최대 하나의 부모를 가질 수 있으므로 여기에 전달된 개체의 현재 부모는 제거됩니다.

`.attach ( object : Object3D )` : 이

객체의 세계 변환을 유지하면서 객체를 이것의 하위로 추가합니다.

`.remove ( object : Object3D, ... ) `: this

이 개체의 하위 개체를 제거합니다. 임의의 수의 객체가 제거될 수 있습니다.

`.removeFromParent ()` : 이

현재 상위 개체에서 이 개체를 제거합니다.

`.clone(재귀: 부울)`: Object3D

재귀 - true인 경우 객체의 하위 항목도 복제됩니다. 기본값은 true입니다.

`.copy (객체 : Object3D, 재귀 : Boolean )` : this

재귀 - true인 경우 객체의 하위 항목도 복사됩니다. 기본값은 true입니다.

`.getObjectByProperty( 이름 : 문자열, 값 : 임의 )` : Object3D

name - 검색할 속성 이름입니다.

value -- 주어진 속성의 값.

`.getObjectById ( id : Integer )` : Object3D

id -- 객체 인스턴스의 고유 번호

객체 자체부터 시작하여 객체와 해당 하위 항목을 검색하고 일치하는 ID가 있는 첫 번째 항목을 반환합니다.

> ID는 시간순으로 1, 2, 3, ...으로 할당되며 새 개체마다 1씩 증가합니다.

`.getWorldPosition( 대상 : Vector3 )` : Vector3

target - 결과가 이 Vector3에 복사됩니다.

`.getWorldQuaternion ( target : Quaternion )` : 쿼터니언

target — 결과가 이 쿼터니언에 복사됩니다.

`.getWorldScale( 대상 : Vector3 )` : Vector3

target - 결과가 이 Vector3에 복사됩니다.

`.getWorldDirection( 대상 : Vector3 )` : Vector3

target - 결과가 이 Vector3에 복사됩니다.

`.rotateOnAxis ( axis : Vector3, angle : Float )` : this

축 - 객체 공간의 정규화된 벡터입니다.

각도 - 라디안 단위의 각도입니다.

`.rotateOnWorldAxis( axis : Vector3, angle : Float)` : this

축 - 월드 공간의 정규화된 벡터입니다.

각도 - 라디안 단위의 각도입니다.

`.traverse ( 콜백 : 함수 )` : 정의되지 않음

콜백 - object3D 객체를 첫 번째 인수로 사용하는 함수입니다.

`.lookAt ( x : 부동, y : 부동, z : 부동 )` : 정의되지 않음

벡터 - 월드 공간에서의 위치를 나타내는 벡터입니다.

선택적으로 월드 공간 위치의 x, y 및 z 구성요소입니다.

월드 공간의 한 점을 향하도록 객체를 회전합니다.

이 방법은 균일하지 않은 크기의 상위 개체가 있는 개체를 지원하지 않습니다.

`.raycast( raycaster : Raycaster, intersects : Array )` : 정의되지 않음

투사된 광선과 이 객체 사이의 교차점을 얻는 추상(빈) 메서드입니다. Mesh, Line 및 Points와 같은 하위 클래스는 레이캐스팅을 사용하기 위해 이 메서드를 구현합니다.
