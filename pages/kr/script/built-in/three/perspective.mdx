# Perspective cameras

> 일반적으로 사용되는 카메라 클래스입니다.

Perspective Camera는 Object3d에서도 상속되므로 아래 설명된 것 외에도 Object3d에 존재하는 속성과 기능을 사용할 수 있습니다.

## Properties

`.fov` : 부동

카메라 절두체 수직 시야(아래에서 위쪽까지)(도). 기본값은 50입니다.

`.far` : 부동
카메라 절두체 원거리 평면. 기본값은 2000입니다.

`.near` : 부동

평면 근처의 카메라 절두체. 기본값은 0.1입니다.

## Methods

`.updateProjectionMatrix ()` : 정의되지 않음

카메라 투영 매트릭스를 업데이트합니다. 매개변수를 변경한 후에는 호출해야 합니다.
