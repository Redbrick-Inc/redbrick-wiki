# MeshStandardMaterial

> 메쉬에 적용할 수 있는 재질(Material) 중 하나입니다. 다양한 다른 재질 클래스들도 있습니다.

## 속성

### .color

`.color` : Color

재질의 색상을 나타냅니다. 기본값은 흰색(0xffffff)입니다.

### .map

`.map` : 텍스처

색상 맵입니다. 선택적으로 알파 채널을 포함할 수 있으며, 일반적으로 `.transparent`나 `.alphaTest`와 결합됩니다.  
기본값은 null입니다.  
텍스처 맵의 색상은 확산(diffuse) `.color`에 의해 조절됩니다.
