# Sprite

> 항상 카메라를 향하는 평면을 정의하는 클래스입니다.

GUI, 파티클 등을 위해 Studio 내부에서 사용되는 요소입니다. 스프라이트도 Object3d에서 상속되므로 아래 설명된 것 외에도 Object3d에 존재하는 속성과 기능을 사용할 수 있습니다.

`.material` : 스프라이트 머티리얼

객체의 모양을 정의하는 SpriteMaterial의 인스턴스입니다. 기본값은 흰색 SpriteMaterial입니다.

`.center` : 벡터2

스프라이트의 기준점 및 스프라이트가 회전하는 지점입니다. (0.5, 0.5) 값은 스프라이트의 중간점에 해당합니다. (0, 0) 값은 스프라이트의 왼쪽 하단 모서리에 해당합니다. 기본값은 (0.5, 0.5)입니다.
