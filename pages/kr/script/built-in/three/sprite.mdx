# Sprite

> 항상 카메라를 향하는 평면을 정의하는 클래스입니다.

GUI, 파티클 등을 위해 Studio 내부에서 사용되는 요소입니다. 스프라이트도 Object3D에서 상속되므로 아래 설명된 것 외에도 Object3D에 존재하는 속성과 기능을 사용할 수 있습니다.

## 속성

### .material

`.material` : SpriteMaterial

Sprite의 외관을 정의하는 SpriteMaterial의 인스턴스입니다. 기본값은 흰색 SpriteMaterial입니다.

### .center

`.center` : Vector2

Sprite의 앵커 포인트로, 이를 중심으로 Sprite가 회전합니다.  
(0.5, 0.5)는 Sprite의 중간 지점을 나타내며, (0, 0)은 Sprite의 좌측 하단 모서리를 나타냅니다.  
기본값은 (0.5, 0.5)입니다.
