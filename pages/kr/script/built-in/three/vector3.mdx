# Vector3

> 3차원 벡터 클래스. 다양한 벡터 연산 등을 제공합니다.

## Properties

x - 이 벡터의 x 값. 기본값은 0입니다.

y - 이 벡터의 y 값. 기본값은 0입니다.

z - 이 벡터의 z 값. 기본값은 0입니다.

## Methods

`.add ( v : Vector3 ) `: 이

이 벡터에 v를 추가합니다.

`.angleTo ( v : Vector3 )` : 부동

이 벡터와 벡터 v 사이의 각도를 라디안 단위로 반환합니다.

`.distanceTo ( v : Vector3 )` : 부동

이 벡터에서 v까지의 거리를 계산합니다.

`.clone ()` : Vector3

이 값과 동일한 x, y 및 z 값을 가진 새 벡터3을 반환합니다.

`.copy ( v : Vector3 )` : 이

전달된 벡터3의 x, y 및 z 속성 값을 이 벡터3에 복사합니다.

`.length ()` : 부동소수점

(0, 0, 0)부터 (x, y, z)까지 유클리드 길이(직선 길이)를 계산합니다.

`.multiplyScalar ( s : Float )` : 이

이 벡터에 스칼라 s를 곱합니다.

`.normalize ()` : 이

이 벡터를 단위 벡터로 변환합니다. 즉, 방향은 같지만 길이가 1인 벡터와 동일하게 설정합니다.

`.set ( x : Float, y : Float, z : Float )` : this

이 벡터의 x, y 및 z 구성요소를 설정합니다.
