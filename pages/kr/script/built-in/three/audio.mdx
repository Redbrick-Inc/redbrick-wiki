# Audio

> 이 클래스는 오디오와 관련된 속성 및 메서드에 대한 액세스를 제공합니다.

## Properties

`.isPlaying` : 부울

오디오가 현재 재생 중인지 여부입니다.

## Methods

`.getPlaybackRate ()` : 부동

PlaybackRate의 값을 반환합니다.

`.getVolume(값)`: 부동 소수점

현재 볼륨을 반환합니다.

`.play(지연) `: 이

hasPlaybackControl이 true이면 재생을 시작합니다.

`.pause ()` : 이

hasPlaybackControl이 true이면 재생을 일시 중지합니다.

`.setPlaybackRate ( value : Float )` : this

hasPlaybackControl이 활성화된 경우 재생 속도를 값으로 설정합니다.

`.setVolume ( value : Float )` : 이

볼륨을 설정하세요.

`.stop ()` : 이

hasPlaybackControl이 활성화된 경우 재생을 중지합니다.

<Callout type="info">
    이외에도, [THREE.Audio](https://threejs.org/docs/?q=audio#api/en/audio/Audio)의 다양한 속성과 메소드를 사용할 수 있습니다.
</Callout>
