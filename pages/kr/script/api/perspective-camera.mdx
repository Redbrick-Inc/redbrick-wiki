# Perspective Camera

> 카메라 객체에 추가되는 API들.

## 메소드

### .activate()

해당 카메라로 전환합니다.  

### .useFPS()

`.useFPS () ` : undefined

카메라를 FPS 모드(1인칭)로 설정합니다.

<br/>
<center>
![scriptlist](../../../../images/useFPS.gif) 1인칭 화면
</center>

### .useTPV()

`.useTPV () ` : undefined

카메라를 3인칭 모드로 설정합니다.

<br/>
<center>
![scriptlist](../../../../images/useTPV.gif) 3인칭 화면
</center>

<Callout type="warning">
    `.useFPS()`와 `useTPV()`는  아바타에 `.setFollowingCamera()` 메소드가 적용되어 있어야 작동합니다.  
    
    Ex) 사용 예시
    ```js showLineNumbers copy filename="PresetScript"
    const avatar = REDBRICK.AvatarManager.createDefaultAvatar();
    const camera = WORLD.getObject("MainCamera");
    avatar.setFollowingCamera(camera); //아바타에 해당 메소드가 적용되어 있어야 합니다.
    avatar.setDefaultController();

    camera.useFPS(); //혹은 camera.useTPV();
    ```
</Callout>

### .useVR()

`.useVR({ VRObject: object }) ` : undefined  

카메라를 VR모드로 설정합니다.  
VR 옵션에 대한 자세한 내용은 [이곳](../../engine/content-type#vr-option)을 참고하세요.  

- object : [Avatar](../api/avatar.mdx) 객체 또는 [Object3D](../built-in/three/object3d.mdx) 객체
