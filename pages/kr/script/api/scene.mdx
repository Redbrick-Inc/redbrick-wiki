# Scene

> 씬에 추가되는 API들.

## 메소드

### .getObject()

`.getObject(title) `

주어진 이름과 일치하는 씬 내의 객체를 찾아서 반환합니다.

- title : 찾고 있는 객체의 이름.

```js /getObject/
const object = WORLD.getObject("objectTitle");
const gui = GUI.getObject("guiTitle");
```
