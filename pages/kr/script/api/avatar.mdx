# Avatar

> 플레이어의 아바타에 추가되는 API.

## 메소드

### spawn(target)

아바타를 지정한 위치로 이동시킵니다.

- target : 객체 또는 객체의 제목. 해당 객체의 위치로 스폰합니다. 기본값은 StartLocation입니다.

### changePlayerSpeed(speed)

아바타의 속도를 변경합니다.

- speed : 아바타의 속도, 기본값은 1입니다.

### changeAxisSpeed(x, y, z)

플레이어 컨트롤과 독립적인 기본 속도를 아바타에 부여합니다.

- x : x축 방향으로의 초당 속도.
- y : y축 방향으로의 초당 속도.
- z : z축 방향으로의 초당 속도.

### changePlayerJumpHeight(height)

아바타의 점프 높이를 변경합니다.

- height : 아바타의 점프 높이, 기본값은 3입니다.
