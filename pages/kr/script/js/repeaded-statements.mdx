# 반복문 사용 방법

> 자바스크립트에서 반복문을 사용하는 방법을 설명합니다.

자바스크립트에서는 for, for ... in, for ... of 문 등과 같은 반복문을 사용할 수 있습니다. 이 페이지에서는 가장 기본적인 for를 사용하여 루프를 설명합니다.

기본적으로 for 블록은 세 가지 표현식을 요구합니다.

- 반복 실행 전에 수행되는 표현식
- 반복문 코드가 실행되는 조건에 대한 표현식
- 반복문 코드 실행 후의 표현식

다음 세 가지 표현식을 사용하여 반복문을 생성할 수 있습니다.

```js
for (expression 1; expression 2; expression 3) {
// code block to be executed
}
```

예를 들어, 다음과 같은 기본 반복을 사용할 수 있습니다.

```js showLineNumbers copy
for (let i = 0; i < 10; i++) {
  console.log("Current i: ", i);
}
```

이 경우, 반복문을 실행하기 전에 i라는 변수에 0을 할당하고, 코드가 실행될 때마다 i의 값을 1씩 증가시킵니다. i가 10보다 작은 경우에만 이 작업을 반복합니다.
