# 이벤트 API

|   API 형식   | 연결된 블록 |                              기능                              |
| :----------: | :---------: | :------------------------------------------------------------: |
|   충돌하기   |    객체     |        특정 대상이 객체와 충돌했을 때 콜백 함수를 실행         |
| 한 번만 충돌 |    객체     |   특정 대상이 객체와 충돌했을 때 콜백 함수를 단 한 번만 실행   |
|  충돌 종료   |    객체     |         충돌 후 대상이 객체에서 떠날 때 콜백 함수 실행         |
|     접촉     |    객체     | 대상이 충돌 후 객체와 접촉 중일 때 콜백 함수를 반복적으로 실행 |

## 충돌하기

특정 대상이 객체와 충돌했을 때 들여쓰기된 코드를 실행합니다.

- `객체` : 충돌 여부를 결정할 객체를 선택합니다.
