# 일반 API

| API 형식 | 연결된 블록 |                 기능                  |
| :------: | :---------: | :-----------------------------------: |
|   대기   |    숫자     | 지정된 시간 동안 코드 실행을 지연시킴 |

## 대기

### 정의

지정된 시간 동안 코드 실행을 지연시킵니다.

- `숫자`: 코드 실행을 지연시킬 초 수를 입력하세요.
