# 회전 API

| API 형식 | 연결된 블록 |                기능                 |
| :------: | :---------: | :---------------------------------: |
| 회전하기 |      x      | x축에 따라 지정한 만큼 객체를 회전  |
|          |      y      | y축에 따라 지정한 만큼 객체를 회전  |
|          |      z      | z축에 따라 지정한 만큼 객체를 회전  |
|  돌리기  |      x      | x축에서 지정된 각도만큼 객체를 돌림 |
|          |      y      | y축에서 지정된 각도만큼 객체를 돌림 |
|          |      z      | z축에서 지정된 각도만큼 객체를 돌림 |

## 회전하기

지정된 방향에서 지정된 양만큼 객체를 즉시 회전시킵니다.

`x` : x축의 회전량을 입력하세요.

`y` : y축의 회전량을 입력하세요.

`z` : z축의 회전량을 입력하세요.

`up` : [회전하기] [y] [양수]와 유사한 결과가 나타납니다.

`down` : [회전하기] [y] [음수]와 유사한 결과가 나타납니다.

`left` : [회전하기] [x] [음수]와 유사한 결과가 나타납니다.

`right` : [회전하기] [x] [양수]와 유사한 결과가 나타납니다.

## 돌리기

입력된 값에 따라 지정된 방향으로 객체를 천천히 회전시킵니다.

`x` : x축 회전 횟수를 입력하세요.

`y` : y축 회전 횟수를 입력하세요.

`z` : z축 회전 횟수를 입력하세요.

`up` : [돌리기] [y] [양수]와 같은 결과가 나타납니다.

`down` : [돌리기] [y] [음수]와 같은 결과가 나타납니다.

`left` : [돌리기] [x] [음수]와 같은 결과가 나타납니다.

`right` : [돌리기] [x] [양수]와 같은 결과가 나타납니다.
