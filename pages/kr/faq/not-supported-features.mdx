# 지원되지 않는 기능

현재 다음 기능은 지원되지 않습니다. 곧 추가될 수 있으니 기대해주세요! 😀

<Steps>

### 글자 크기 조정

현재 글자 크기를 줄이는 기능은 제공되지 않습니다.

### GUI에서 GIF 사용

GIF를 추가하여 GUI로 사용할 수 없습니다. 그러나 `tween` 또는 `move` 기능을 사용하여 GUI를 애니메이션으로 만들 수 있습니다.

### 제어 키 사용자 지정

아바타의 제어 키를 'WASD' 이외의 다른 키로 변경할 수 없습니다.

### 이전 코드 접근

Git과 같은 방식으로 이전 코드를 추적하는 기능은 없습니다.

<Callout type="info">
  이전 코드를 보관하고 싶다면 파일로 저장하여 컴퓨터에 보관할 수 있습니다.
  <br />
  <center>![파일로 저장](../../../images/save_to_file1.png) 파일로 저장</center>
</Callout>

### 스크립트 위치 이동

스크립트의 위치를 이동시키는 기능은 현재 지원되지 않습니다.

### 사용자 정의 글꼴 추가

자신의 글꼴을 추가하는 기능은 현재 지원되지 않습니다.

### GUI 호버 기능

GUI 호버 기능은 제공되지 않습니다.

</Steps>
