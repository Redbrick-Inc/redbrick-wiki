# Replacing Assets

> This section provides information on replacing assets.

---

### Types of Assets

There are two main types of assets.

1. **3D Objects**: Objects like Camera, Spawn, Light, Effect, Mesh, Sound, and Video fall under this category.
   <br/>They have Transform properties such as position, rotation, and scale.

2. **GUI Objects**: GUI objects are included here. 
   <br/>They have Transform properties such as center, anchor, offset, and size.

<br/>
<center>![replace-asset_asset-type](/images/engine/replace-asset/replace-asset_asset-type.png) Types of Assets</center>

<Callout type="error">
    You cannot replace objects of different types.
    <br/>Currently, Sound and Video cannot be replaced with any other type of asset. This will be added in the future.
</Callout>
<br/>

---

### Replacing Assets

First, the properties that will be retained when replacing assets are as follows:
- Common
    - title
    - Parent
- 3D Objects
    - Object Transform
- GUI Objects
    - GUI Transform: anchor, offset(px)
    - Text: textInput, fontSize, fontColor

<br/>
<center>![replace-asset_retained-property](/images/engine/replace-asset/replace-asset_retained-property.png) Properties Retained During Asset Replacement</center>

<br/>
### Replacement Steps
<Steps>
    ### Select the Asset to Replace

    You can select the asset you want to replace from the Explorer tab or the Scene panel.

    <br/>
    <center>![replace-asset_step-1](/images/engine/replace-asset/replace-asset_step-1.png) Select the Asset to Replace</center>

    ### Click More (...) on the Asset to be Replaced

    In the asset library or My Assets, select the asset to be replaced and click on More.

    <br/>
    <center>![replace-asset_step-2](/images/engine/replace-asset/replace-asset_step-2.png) Click More on the Asset</center>

    ### Click Replace to Replace the Asset
    The replacement is complete! If it does not succeed, please refer to the <Link href="#types-of-assets"><u>related information</u></Link>.

    <br/>
    <center>![replace-asset_step-3](/images/engine/replace-asset/replace-asset_step-3.png) Click Replace</center>
</Steps>
