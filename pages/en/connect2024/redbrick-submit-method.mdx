### Add the project link
<br />
  <center>![](/images/connect2024/image_(22).png)</center>

1. The project link here is the link to the project you uploaded to Redbrick earlier. 
Click on publishing (1st) and then click on your created project (3rd). 
At this time, make sure that the 2nd button is set to **Public** to make your project visible.

<br />
  <center>![](/images/connect2024/image_(23).png)</center>

2. The address below is the project link you're submitting.
<br />
  <center>![](/images/connect2024/image_(24).png)</center>

3. Copy this link and paste it as shown below.
<br />
  <center>![](/images/connect2024/image_(25).png)</center>

---

### Upload the project file
<br />
  <center>![](/images/connect2024/image_(26).png)</center>

1. You need to upload the Unity WebGL file you saved earlier.
First, go to the location where you saved the file in Unity, right-click on the file, and select the compress option.
<br />
  <center>![](/images/connect2024/image_(27).png)</center>

2. When compressed correctly, the file will look like this:
<br />
  <center>![](/images/connect2024/image_(28).png)</center>

3. Click Add file below,
<br />
  <center>![](/images/connect2024/image_(29).png)</center>

4. A window like the one below will appear, click the Browse button here.
<br />
  <center>![](/images/connect2024/image_(30).png)</center>

5. Select the compressed file from earlier and click open,
<br />
  <center>![](/images/connect2024/image_(30)-1.png)</center>

6. It will briefly appear like this,
<br />
  <center>![](/images/connect2024/image_(30)-2.png)</center>

7. Then you can confirm that the file has been successfully uploaded as shown below.
<br />
  <center>![](/images/connect2024/image_(30)-3.png)</center>
