# Return to original position when it falls

<Tabs items={["example 1", "example 2"]} defaultIndex="0" >
    
    <Tabs.Tab>

        In this example you can make it by simply checking `PLAYER`'s `Y` position.

        ```js showLineNumbers copy
        function Update(dt) {
            //respawn when PLAYER y position is less than -5
            if (PLAYER.position.y < -5){
                PLAYER.spawn();
            }
        }
        ```

    </Tabs.Tab>

    <Tabs.Tab>

        In this example you can make it by colliding to a mesh under your world.
        <Steps>

            ### Place object under your world
            Place the mesh where you want it and resize it. Check Body, and make sure Anchored is checked.

            <br/>
            <center>![respawn_step-1](/images/snippet/redbrick-extension/respawn/respawn_step-1.png) Place mesh, and check body & anchored </center>

            ### Make mesh transparent

            Use the Opacity to adjust the transparency.

            <br/>
            <center>![respawn_step-2](/images/snippet/redbrick-extension/respawn/respawn_step-2.png) Change Opacity </center>

            ### Create a place to return to its original location

            Click the Spawn that you want to create -> place a StartLocation

            <br/>
            <center>![respawn_step-3](/images/snippet/redbrick-extension/respawn/respawn_step-3.png) Create and place spawn location </center>

            ### Insert Code

            ```js showLineNumbers copy
            const spawnbox = WORLD.getObject("BOX(b5d)");

            function Start() {
                // use here spawn related logic
                spawnbox.onCollide(PLAYER, () => {
                    PLAYER.spawn();
                });
            }
            ```

        </Steps>
