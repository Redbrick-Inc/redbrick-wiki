# How to use `PLAYER` properly

<Steps>

### use PLAYER

```js showLineNumbers copy filename="Collect coin"
const coin = WORLD.getObject("coinTitle");

coin.onCollide(PLAYER, () => {
  coin.kill();
});
```

### PLAYER in `Update(dt)`

```js showLineNumbers copy filename="PresetScript" {1, 6, 15, 17}
function Update(dt) {
  //respawn when avatar y position is less than -5
  if (PLAYER.position.y < -5) {
    PLAYER.spawn();
  }
}
```

</Steps>
