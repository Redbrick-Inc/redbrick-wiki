# MathUtils

> A module that provides mathematical functions. It can be used as MathUtils.abs().

### .abs()

`.abs(number) `

Returns a given number of absolute values.

### .round()

`.round(number, digit) `

Returns the value by rounding the given number. digit means how many decimal places it represents.

### .ceil()

`.ceil(number, digit) `

Returns the value by raising the given number. digit means how many decimal places it represents.

### .floor()

`.floor(number, digit) `

Returns the value by decreasing the given number. digit means how many decimal places it represents.

### .pow()

`.pow(number, exponent) `

Returns the value calculated by exponent below the given number.

### .sqrt()

`.sqrt(number) `

Returns a given number of square root values.

### .log()

`.log(number, base) `

Returns the log value calculated with base below.

### .getRandom()

`.getRandom(min, max) `

min, returns a random integer within the max range.
