# MeshStandardMaterial

> This is one type of material that can be applied to meshes. There are many other material classes.

<Callout type="info">
    The MeshStandardMaterial also inherits from Material, so in addition to the properties and functions described below, you can also use the properties and functions that exist in [THREE.Material](https://threejs.org/docs/?q=mate#api/en/materials/Material) and [THREE.Material.MeshStandardMaterial](https://threejs.org/docs/?q=MeshStandardMaterial#api/en/materials/MeshStandardMaterial).
</Callout>

## Properties

### .color

`.color` : Color

Color of the material, by default set to white (0xffffff).

### .map

`.map` : 텍스처

The color map. May optionally include an alpha channel, typically combined with `.transparent` or `.alphaTest`  
Default is null.  
The texture map color is modulated by the diffuse `.color`
